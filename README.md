# user-service

## Set up
### Create virtual environment
```buildoutcfg
cd user-service
python3 -m venv .env
source .env/bin/activate
```
### Install dependencies
```buildoutcfg
pip install -r requirements.txt
```

## Run on localhost
```buildoutcfg
cd user_service
python app.py
```

## Run in Docker
```buildoutcfg
docker-compose up
```
